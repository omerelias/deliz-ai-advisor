AI-powered chat advisor for WooCommerce delicatessen shops. Multilingual product advice powered by Claude.

== Description ==

Deliz AI Advisor adds an intelligent chat bubble to your WooCommerce product pages. Visitors get instant, expert-level answers about cooking methods, cuts, storage, and pairings — in Hebrew, Russian, Arabic, or English.

**Features:**

* Chat bubble on product pages (fully customizable)
* Multilingual auto-detection (HE / RU / AR / EN)
* Product-aware: knows the current product, price, and related items
* Response cache — repeated questions are free
* Per-IP rate limiting + daily USD cap
* Conversation log + statistics dashboard
* Conversion tracking (add-to-cart within 30 min of chat)
* BYOK — use your own Anthropic API key

**Requirements:**

* WooCommerce (active)
* An Anthropic API key with credits

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate through the `Plugins` menu.
3. Go to `Deliz AI → Settings`, enter your Anthropic API key, and click `Test Connection`.
4. Visit any product page — the chat bubble should appear in the corner.
